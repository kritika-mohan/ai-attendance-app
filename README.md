# 📸 AI Attendance App

A React Native mobile application designed to modernize attendance tracking using smart verification techniques like geotagging, network validation, and biometric concepts.

This project was developed as a solution to a Smart India Hackathon (SIH) problem statement, aiming to replace traditional register-based attendance systems with a more secure and automated approach.

---

## 🚀 Features

- 📱 Mobile-based attendance system  
- 📍 Geotagging with timestamp  
- 🌐 WiFi / MAC address-based verification  
- 🎙️ Speaker recognition (concept-based)  
- 👤 Face recognition using depth detection (concept/proposed)  
- ⚡ Simple and intuitive user interface  

---

## 🧠 Proposed Solution

The application follows a multi-layer verification pipeline to ensure accurate and tamper-proof attendance:

1. Geotagging with Timestamp – verifies user location  
2. Network Verification – checks WiFi/MAC authenticity  
3. Speaker Identification – voice-based authentication  
4. Face Recognition (Depth Detection) – prevents spoofing  

This layered approach improves reliability and minimizes proxy attendance.

## ⚙️ Installation & Setup

### 1. Clone the repository

git clone https://github.com/kritika-mohan/ai-attendance-app.git  
cd ai-attendance-app  

### 2. Install dependencies

npm install  

### 3. Run the application

For Android:

npm run android  

For iOS:

npm run ios
